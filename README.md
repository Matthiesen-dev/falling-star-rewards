# Falling Star Rewards

<div>
  <img src="https://mods.matthiesen.dev/badges/matthiesenLibAPI.svg" alt="Matthiesen Lib API">
</div>

A Server-side mod that adds falling stars to the world, which can drop a variety of items where they land. The mod is designed 
to be highly configurable, allowing server owners to customize the types of items that can drop, the frequency of falling stars, 
and other aspects of the gameplay experience.

Stars will fall from the sky at random intervals, and when they land, they will create a small explosion and drop items based on the configured events. 
The items that can drop from the falling stars can be customized to include anything from common resources to rare and valuable items, with support for 
custom model data, and custom NBT data! Making it an exciting addition to any Minecraft server.

To get started, simply drop the mod into your server's mod folder start the server once to generate the default configuration files. From there, you 
can customize the configuration files to your liking and start enjoying the rewards of the falling stars in your Minecraft world!
Check the docs link below for more information on how to customize the mod and make the most out of its features.
